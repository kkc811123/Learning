📝 简介
本项目提供了一系列精心整理的 Surge 规则文件，用于：
	•	🚀 网络加速与优化
	•	🔒 广告拦截
	•	📱 应用分流
	•	🌍 地区访问控制
🚀 快速开始
使用方法
	1.	在 Surge 中添加规则集
	2.	复制下方对应的规则链接
	3.	粘贴到 Surge 配置中的 [Rule] 部分
规则文件
# 示例配置

[Rule]
# 在这里添加你的规则文件链接
RULE-SET,https://raw.githubusercontent.com/kkc811123/Surge-Ruleset/main/你的规则文件.list,POLICY-NAME
